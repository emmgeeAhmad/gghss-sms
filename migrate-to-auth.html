One-Time Security Migration

Run this once, after enabling Email/Password sign-in in Firebase Console, and before replacing the main app file with the new secure version.

This tool creates a real, secure login (Firebase Authentication) for every existing student and staff account, using each person's current password as their starting password — so nobody's login changes from their point of view. It then removes the old plain-text password from the shared database, since it's no longer needed.

Staff need a real email address (for "Forgot password" to work going forward) — enter one for each person below. Students don't need one; a private technical placeholder is generated automatically for them.

Safe to leave this page and come back — anyone already migrated is automatically skipped if you run it again.
