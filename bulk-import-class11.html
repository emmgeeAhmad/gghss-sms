Bulk Import — Class 11 Admissions (2026)

This is a one-time import tool, completely separate from the main school app file — it does not modify gghss-singhpora-kalan.html in any way. It connects to the same shared database and adds these students directly to the Admission Register, the same as if they'd been entered one by one through the app's own "New Admission" form.

Students already in the system (matched by Admission No.) are automatically detected below and skipped, so it's safe to run even after you've already manually added a few of them.
